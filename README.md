# Hello, World!

I have taken liberty to create a repository for the simple "Hello, World!" program every programming language which I know of, and more, to touch new areas I have not yet explored.

Some languages need to be separated into subdirectories, especially Go, because Go does not like having foreign source files mixed in the same folder as normal Go code. I may also sort languages into folders such as C, if they fit into the C family, or SQL, if they are SQL files

I will obviously add more files, there is a bunch of languages I will try out for this, Hello World is a simple thing that has been used as samples for a long time in programming, and this is the simplist way to do all of that in one repository.
